# ユーザー種別・ロール仕様

## グローバルロール

システム全体で持つロール。


| ロール       | 説明                   |
| --------- | -------------------- |
| `ADMIN`   | システム管理者。教師アカウントを発行する |
| `TEACHER` | 授業を作成・管理する教員・講師      |
| `STUDENT` | 課題を提出する学生。自己登録で作成    |


### アカウント作成

- `STUDENT` — メールアドレス＋パスワードで自己登録
- `TEACHER` — `ADMIN` が管理画面から発行
- `ADMIN` — 初期データとして登録（シードデータ）

---

## 授業内ロール（Course Role）

授業への参加時に持つロール。グローバルロールとは独立。


| ロール       | 説明             |
| --------- | -------------- |
| `TEACHER` | 授業のオーナー。全操作が可能 |
| `TA`      | 採点・フィードバックのみ可能 |
| `STUDENT` | 課題の閲覧・提出のみ可能   |


- `TEACHER` は自分が作成した授業において授業内ロール `TEACHER` を持つ
- `STUDENT` は授業コードで参加後、授業内ロール `STUDENT` を持つ
- `TEACHER` が授業内の `STUDENT` を `TA` に昇格できる

---

## 権限マトリクス

### システム操作


| 操作         | ADMIN | TEACHER | STUDENT |
| ---------- | ----- | ------- | ------- |
| 教師アカウントの発行 | ✓     | -       | -       |
| 授業の作成      | -     | ✓       | -       |
| 自己登録       | -     | -       | ✓       |


### 授業操作（授業内ロール基準）


| 操作              | TEACHER (owner) | TA  | STUDENT |
| --------------- | --------------- | --- | ------- |
| 授業の編集           | ✓               | -   | -       |
| 授業コードの再発行       | ✓               | -   | -       |
| 生徒の追加・削除        | ✓               | -   | -       |
| 生徒をTAに昇格        | ✓               | -   | -       |
| 課題の作成・編集        | ✓               | -   | -       |
| 課題の公開設定         | ✓               | -   | -       |
| 課題の返却           | ✓               | -   | -       |
| 全提出の閲覧          | ✓               | ✓   | -       |
| 採点・スコア入力        | ✓               | ✓   | -       |
| フィードバックの追加      | ✓               | ✓   | -       |
| 課題の閲覧（公開済み）     | ✓               | ✓   | ✓       |
| 課題の提出           | -               | -   | ✓       |
| 自分の提出結果の閲覧（返却後） | -               | -   | ✓       |
| 成績一覧の閲覧         | ✓               | ✓   | 自分のみ    |
